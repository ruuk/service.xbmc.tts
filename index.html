View project onGitHub

Installation

Windows users, click the link to the right to download the XBMC TTS installer. It will download and install XBMC (if not already installed) and it will download and install the newsest version of the XBMC TTS addon and it's requirements.

Others should install XBMC (if necessary) and then install the repository and install XBMC TTS from within XBMC. Links are on the right.
Android and iOS will not work currently.
